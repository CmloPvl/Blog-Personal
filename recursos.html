Mis Recursos Recomendados

Aquí comparto algunas de las plataformas, sitios web y herramientas que me han ayudado a aprender desarrollo web, programación y ciberseguridad.

  • Road Map de manzdev - Manz.dev es la web personal de J. Román (ManzDev), donde comparte recursos, herramientas, proyectos y consejos prácticos sobre desarrollo web. Además, incluye acceso a su Roadmap de JavaScript, guías de CSS, utilidades para desarrolladores y enlaces a su contenido en Twitch y YouTube.
  • YouTube - Plataforma donde sigo canales y tutoriales sobre desarrollo web, programación y ciberseguridad. Encuentro contenido práctico, guías paso a paso y explicaciones visuales que complementan mi aprendizaje como futuro Full Stack y hacker ético.
  • Boostrap - Framework de código abierto para diseño web, que permite crear sitios y aplicaciones responsive de forma rápida. Ofrece un sistema de rejilla (Grid), componentes predefinidos y utilidades CSS listas para usar, facilitando el desarrollo de interfaces modernas y adaptables.
  • ChatGPT - Herramienta que utilizo para resolver dudas, generar ideas y practicar conceptos de programación.
  • Academia Desafío Latam - Desafío Latam es una academia de formación tecnológica online que ofrece programas intensivos y prácticos en áreas como desarrollo web, data science, diseño UX/UI y ciberseguridad. Su enfoque está en preparar a los estudiantes para el mundo laboral mediante proyectos reales, clases en vivo y apoyo constante. Actualmente estoy cursando el programa Full Stack JavaScript en Desafío Latam, donde estoy aprendiendo a construir aplicaciones completas usando tecnologías modernas como JavaScript, Node.js y React, fortaleciendo mis habilidades para crecer como desarrollador.
